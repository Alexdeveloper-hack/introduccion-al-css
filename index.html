1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS define cómo se deben mostrar los elementos en la pantalla, en papel, o en otros medios.

2. ¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido de un documento (escrito en HTML) de su presentación. Esto permite que los diseñadores web cambien el estilo de una página web sin afectar su estructura.

3. ¿Cómo se puede incluir CSS en un documento HTML?

CSS se puede incluir en un documento HTML de tres maneras: Inline: Usando el atributo style en una etiqueta HTML

Interno: Usando una etiqueta stylo dentro del documento

Externo: Enlazando a un archivo CSS externo con la etiqueta

. ¿Qué son los selectores en CSS?

Los selectores en CSS son patrones que se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos. Ejemplos de selectores incluyen:

Elementos: p selecciona todos los elementos

.

Clases: .clase selecciona todos los elementos con la clase clase.

ID: #id selecciona el elemento con el ID id.

5. ¿Qué son las propiedades y valores en CSS?

Las propiedades son características que se pueden modificar en un elemento HTML, como color, font-size, o margin. Los valores son los ajustes que se aplican a estas propiedades.

6. ¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja (box model) describe cómo se representa un elemento en CSS. Cada elemento se considera una caja que consta de:

Contenido: El texto o imágenes dentro del elemento.

lleno (padding): El espacio entre el contenido y el borde.

Borde (border): Una línea que rodea el relleno.

Margen (margin): El espacio fuera del borde, separando el elemento de otros elementos.

. ¿Qué es el "cascading" en CSS?

El "cascading" se refiere a la forma en que CSS aplica estilos a los elementos. Si hay múltiples reglas que afectan un elemento, se aplican según un orden de prioridad, que considera la especificida d de los selectores y el orden en que aparecen las reglas. Esto significa que las reglas más específicas y las que aparecen más tarde en el código tendrán mayor prioridad.

8. ¿Qué son las media queries?

Las media queries son una característica de CSS que permite aplicar estilos diferentes según las características del dispositivo, como el ancho de la pantalla. Son fundamentales para el diseño responsivo.

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos comunes incluyen: Color: Cambiar el color del texto.

uentes: Establecer el tipo, tamaño y estilo de fuente.

márgenes y rellenos: Ajustar el espacio alrededor y dentro de los elementos.

Bordes: Agregar bordes a los elementos

Fondo: Establecer colores o imágenes de fondo.

10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

CSS se puede usar para crear un diseño responsivo a través de: Media queries: Para aplicar diferentes estilos en función del tamaño de la pantalla.

Unidades relativas: Usar unidades como %, em, o rem en lugar de píxeles.

Flexbox y Grid: Usar estos sistemas de diseño para crear layouts que se ajusten automáticamente a diferentes tamaños de pantalla.
